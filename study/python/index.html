설치 및 실행
1. Python 2에 대한 지원이 중단되므로 특별한 이유가 없다면 Python 3이상의 최신 버전 설치. https://www.python.org/downloads/
2. 윈도우 Shell에서 파이썬을 곧바로 실행하려면 설치경로(예. C:\Program Files\Python36)를 path 환경변수에 추가.
3. Python Shell은 각 문장을 대화식 인터프리터가 해석하여 결과를 바로 출력해준다.
4. hello.py 실행 : 저장된 디렉터리에서 셸을 통해 python hello.py를 실행.
5. Python Shell에서 import this를 실행하면 파이썬의 철학을 확인할 수 있다.
자료형
1. type() 함수를 통해 인자의 자료형을 확인할 수 있다.
2. 파이썬의 변수는 동적 자료형(실행 도중 자료형이 바뀔 수 있다)으로, 자바와 같은 강한 타입 검사를 하지 않는다. 또, 변수를 나타내는 키워드 없이 식별자만 있으면 변수로써 이용할 수 있다.
3. 복소수도 이용 가능 : x + yj
» str() : 사람이 읽기 편한 형태, repr() : 인터프리터가 인식하는 형태
4. 파이썬의 강력한 내장 자료형 : 리스트, 튜플, 딕셔너리, 셋
» 리스트는 변경 가능 <-> 튜플은 변경 불가능
» 튜플은 불변하므로 딕셔너리의 키로 이용할 수 있다.
문법
1. 주석 : #
2. 라인 유지 : \
3. 조건 분기 : if ~ elif ~ else
4. bool 연산 : and, or, not
5. False로 간주되는 값 : None, 0, 0.0, '', [], (), {}, set()
6. while 반복 : while + 조건식
7. for 순회 : for + 순회 변수
8. 반복문 제어 : break, continue
9. break 호출 확인 : else
10. 콤마로 구분된 값들을 자동으로 튜플로 구성하여 다중 할당, 다중 반환이 가능하다.
순회 대상 생성
1. 수열 생성 : range([start,] stop[, step]). start 기본값 0, step 기본값 1. stop 자체는 미포함
2. 여러 시퀀스 동시 순회 : zip(). 같은 오프셋의 요소들로 튜플을 생성. 시퀀스가 하나라도 끝나면 순회 종료
3. comprehension : 순회를 통해 리스트, 딕셔너리, 셋 생성
함수
1. 정의 : def name():
2. return이 없으면 호출자는 None을 얻는다. None 검사 : is None
3. * : 인자를 튜플로. ** : 인자를 딕셔너리로.
4. docstring : 함수 body 첫 줄에 문자열로 함수 설명. help()로 출력된다.
5. 함수 외부 scope에 접근할 수 있다.
6. 함수 내부에 함수 정의 가능.
7. 람다 함수 : lambda
8. Generator : 자료를 순회할 때마다 다음 항목 반환
» range() : 미리 모든 값을 생성한다.
9. Decorator : 함수 꾸미기. 동적으로 기능 추가.
10. * : 인자 리스트 언패킹, ** : 키워드 인자 딕셔너리 언패킹
11. def f(a: str, b: str = 'hello') -> str: // 타입 명시
네임스페이스
1. 네임스페이스 : "속성 이름 -> 객체 이름" 매핑
2. 메인 프로그램은 전역 네임스페이스를 정의
3. globals() : 전역 네임스페이스 정보를 담은 딕셔너리
4. 함수는 지역 네임스페이스를 정의
5. locals() : 지역 네임스페이스 정보를 담은 딕셔너리
6. global 키워드로 전역 변수 접근
7. 변경 가능한 속성은 del 키워드를 이용해 객체에서 지울 수 있다. (출처: https://docs.python.org/3/tutorial/classes.html)
