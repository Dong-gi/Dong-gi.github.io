문법
변수
1. 자바스크립트는 CamelCase 이용. 변수명에 '_'과 '$'가 허용된다.

2. 느슨한 타입 이용. typeof 연산자로 확인.

3. var : 지역변수 선언 키워드

4. 타입 : Object, String, Number, Boolean, Undefined, Null
» Boolean() : "", 0, NaN, null, undefined => false, 그 외 => true
» Number(), parseInt(), parseFloat(), number.toString(radix)
» (null == undefined) === true
» 문자 리터럴 : \xFF, \uFFFF
연산자
1. 비트 연산자 : ~, &, |, ^, <<, >>, >>>

2. Boolean 연산자 : !, &&, || <- short circuit

3. var n = (1, 2, 3, 4, 5); » n === 5
문장
1. for-in : 객체 프로퍼티 순회

2. labelName: 문장 » outerLoop: for(){ /*~*/ break outerLoop; /*~*/ continue outerLoop; }

3. with (object) { } : 스코프 고정

4. switch : 모든 타입에 대해 동작, case에 상수가 아닌 표현식도 가능
함수
1. 함수는 arguments 객체를 통해 매개변수에 접근.

2. 오버로딩 불가. 매개변수 길이 불일치해도 됨.
주석
<noscript></noscript> : 자바스크립트를 지원하지 않/못하는 경우 출력
