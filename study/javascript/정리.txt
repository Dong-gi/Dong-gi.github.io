1. 자바스크립트란 무엇인가
	1.2 자바스크립트 구현
		자바스크립트 = ECMAScript + DOM + BOM
		Document Object Model : XML을 HTML에서 사용할 수 있도록 확장한 API(Application Programming Interface)
			DOM은 전체 페이지를 노드의 계층 구조로 변환.
		Browser Object Model : 브라우저 창에 접근하고 조작할 수 있게 하는 인터페이스. BOM을 이용해 브라우저에 표시된 페이지와는 별개의 컨텍스트에서 브라우저와 상호작용.

2. HTML 속의 자바스크립트
	2.1 <script> 요소
		자바스크립트를 HTML 페이지에 삽입하는 일차적인 방법.
		* async 속성 : 스크립트를 즉시 내려받지만 자원을 내려받거나 다른 스크립트를 불러오는 등 다른 페이지 작업을 방해해서는 안된다고 지시.
		* src 속성 : 실행할 코드를 포함한 외부 파일의 위치
		* type 속성 : 생략해도 안전

		인라인 자바스크립트 코드에서 문자열 "</script>"는 사용할 수 없다. '/'문자를 "\/"로 이스케이프하면 사용 가능.
		<script>와 </script> 태그 사이에 스크립트 코드가 있고 src 속성도 사용했다면 인라인 코드는 무시.
