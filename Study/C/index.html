C, C++

C is strong.

기본 자료형

1. char(1byte), short(2bytes), int(short <= int <= long), long(4bytes) // unsigned
2. float(4bytes), double(8bytes), long double(10bytes)
3. void : 반환 타입 선언 | 매개변수 선언 | void 포인터

리터럴

1. unsigned : 123U, long : 123L, 16진수 : 0x, 8진수 : 0
2. 지수 표기 : E, float : 1.F

상수

1. const float PI = 3.141592F
2. #define PI 3.141592 // 컴파일러가 단순 치환 작업함

Storage Classes

1. auto : 모든 지역변수 기본값. int a; == auto int a;
2. register : RAM이 아닌 레지스터에 변수 생성.
3. static : 처음 호출 시 한번만 초기화되고 프로그램 종료시까지 유지
4. extern : 외부 파일 전역 변수 참조

연산자

1. 산술 : +, -, *, /, %, ++, --
2. 관계 : ==, !=, >, >=, <, <=
3. 논리 : &&, ||, !
4. 비트 : &, |, ^(xor), ~(not), <<, >>
5. sizeof(), &(address), *(pointer), ?:(condition), (형변환), [], ->

2. 전역변수 초기화값 : 0, '\0', 0.f, 0., NULL

3. int arr[3][3];

구조체

1. struct [식별자] { 멤버변수 목록 }

typedef

1. 기정의된 타입을 다른 이름으로 이용할 수 있게한다.

IO

1. 표준입출력 파일 포인터 : stdin, stdout, stderr
2. int getchar(), int putchar(int c), char* gets(char* s), int puts(const char* s)
3. int scanf(const char* format, ...), int printf(const char* format, ...)

4. int fclose(FILE* fp), int fgetc(FILE* fp), char* fgets(char* buf, int n, FILE* fp), int fputc(int c, FILE* fp), int fputs(const char* s, FILE* fp)
5. binary : size_t fread(void* _Buffer, size_t _ElementSize, size_t _ElementCount, FILE* _Stream)
size_t fwrite(const void* _Buffer, size_t _ElementSize, size_t _ElementCount, FILE* _Stream)
