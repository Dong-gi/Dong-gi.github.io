id가 "p1"인 문단

console.log(document.getElementById('p1'));
console.log(document.getElementById('p1').textContent);

getElementById()로 가져온 객체의 타입은 Element

document.getElementById('p1').textContent = '변경된 텍스트';

document.write('<p>Hello World!</p>');

HTML이 모두 로드된 후에 실행되는 write()는 존재하는 모든 HTML을 지운다.

var p = document.createElement('p');
p.textContent = "new string";
document.getElementById('p1').appendChild(p);
