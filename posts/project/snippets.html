한글 종성 구분
  1. 한글은 0xAC00('가')부터 0xD7A3('힣')까지 총 11,172자가 정의
  2. 11,172 = 초성 19 * 중성 21 * 종성 28(종성 없는 경우 포함)
    3. EastAsianWidth
  3. 코드 포인트 = (초성 인덱스 * 21 + 중성 인덱스) * 28 + 종성 인덱스 + '가'0xAC00
    • (코드 포인트 - '가'0xAC00) % 28 = 종성 인덱스
    • (코드 포인트 - 종성 인덱스 - '가'0xAC00) / 28 % 21 = 중성 인덱스
    • ((코드 포인트 - 종성 인덱스 - '가'0xAC00) / 28 - 중성 인덱스) / 21 = 초성 인덱스
  4. 조사 처리 예시 코드

예. 2020년 2월 22일

  • d = 22 # 22일
  • m = 12 # 1(3월), 2(4월), ..., 12(2월)
  • c = 20 # year // 100
  • y = 19 # (year % 100), 1-2월이면 -1
  • w = (d + floor(2.6 * m - 0.2) - 2 * c + y + (y // 4) + (c // 4)) % 7

    • Week day : 0(일요일), 1(월요일), ..., 6(토요일)

이미지 인코딩/디코딩
  1. 비트열 a, b에 대하여 (a ^ b) ^ b = a임을 이용
  2. 인코딩 ::= 기존 데이터의 각 바이트에 대하여, data[i] ^= mask[i]
  3. 디코딩 ::= 인코딩된 데이터 각 바이트에 대하여, data[i] ^= mask[i]
  4. 따라서 XOR에 이용하는 mask만 정확히 재현할 수 있다면 빠르게 in-place 인코딩/디코딩이 가능하다
