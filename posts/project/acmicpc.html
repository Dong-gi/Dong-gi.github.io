1003 피보나치 함수 > 이산수학

  • 정의
    • let answer(n, 0) ::= fibonacci(n) 호출 시 "0"을 출력하는 횟수
    • let answer(n, 1) ::= fibonacci(n) 호출 시 "1"을 출력하는 횟수
  • 초기값
    • answer(0, 0) == 1, answer(0, 1) == 0
    • answer(1, 0) == 0, answer(1, 1) == 1
  • 점화식

    • fibonacci(n) == fibonacci(n-2) + fibonacci(n-1)이므로,

    • answer(n, 0) == answer(n-2, 0) + answer(n-1, 0)
    • answer(n, 1) == answer(n-2, 1) + answer(n-1, 1)

  • 시점과 종점 모두 행성계 안에 있는 경우 : 지나갈 필요 없음

1011 Fly me to the Alpha Centauri > 이산수학

  • x → y 도달하는 동안 이동하는 수열 (m1, m2, m3, ..., mx) 중에서 불연속적으로 수를 뽑아서,

    • (1, 2, 3, ..., n-1, n, n-1, ..., 3, 2, 1)을 항상 만들 수 있다; 2n - 1번 이동

  • 남은 거리 (y - x - n^2) 중, n으로 이동할 수 있는 건 n으로 이동; (y - x - n^2) / n번 이동
  • 나머지가 0이 아니라면 [1, n-1] 중 하나로 1번 이동

1193 분수찾기 > 이산수학

  • n번째(1부터 카운트) 대각선에 위치하는 분수는 n개
  • n번째 대각선에 위치하는 분수의 (분자) + (분모) == n + 1
  • 홀수 번째 대각선은 n/1, 짝수 번째 대각선은 1/n부터 시작

1260 DFS, BFS > 그래프

  • DFS 탐색 중인 모든 정점은 DFS 탐색을 적용
  • BFS 탐색 중인 모든 정점은 BFS 탐색을 적용; 다음 레벨의 모든 정점을 정렬해서 큐를 구성하는 건 NG

1463 1로 만들기 > 이산수학

  • let P(n) ::= n을 1로 만드는 데 필요한 최소 연산 수
  • 정의에 의해, P(3n) = P(n) + 1, P(2n) = P(n) + 1, P(n+1) = P(n) + 1
  • 1부터 시작하여 정의에 따라 1,000,000까지 한 번씩만 방문하면 모든 답을 구할 수 있다

    • 정의상 P(n)은 최소 연산 수기 때문에, 한 번 P(n)를 계산했다면 다시 계산할 필요가 없다
