이클립스 단축키

  1. Ctrl + Shift + L : 전체 단축키 목록
  2. F3 : 정의로 이동
  3. F4 : Type hierarchy
  4. Ctrl + Shift + G : 참조 위치 검색
  5. Ctrl + E : 열린 파일 목록 팝업
  6. Ctrl + D : 라인 제거
  7. Ctrl + L : 라인 이동
  8. Ctrl + O : Outline 팝업
  9. Alt + ←/→ : 편집 위치 전/후이동
  10. Ctrl + (Shift) + K : 블록 지정 텍스트 (뒤로) 검색
  11. Ctrl + (Shift) + / : 블록 지정 텍스트 (/**/) 주석 토글
  12. Ctrl + (Shift) + W : (전체) 파일 닫기
  13. Ctrl + Shift + F, Ctrl + I : 블록 지정 텍스트에 포맷 적용
  14. Ctrl + 1 : Quick fix
  15. Ctrl + Shift + O : import 정돈
  16. Ctrl + Shift + T : 클래스 검색
  17. Ctrl + Shift + R : 리소스 검색

윈도우

  • 차단한 앱 알림 해제

    • 레지스트리 : HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Notifications\Settings\

  • 확장자 일괄 변경
    • ren *.html *.txt
  • dir 결과 저장
    • dir /s /b *.png > png.txt
    • /s : 재귀
    • /b : "디렉터리/파일명"만 출력
    • > : 저장
  • 시작 프로그램 등록
    1. shell:startup 실행

      2. C:\Users\%USER_NAME%\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup

    2. exe 바로가기 붙여넣기
  • 부트매니저 설정
    • SSD에 설치한 윈도우가 BIOS 설정에서는 검색되지 않는 경우에 대해 해결됨 확인
    • 자료 : https://m.blog.naver.com/pspsim/221141734669
      1. msconfig 실행
      2. 부팅 메뉴에서 기본값 설정, 불필요한 부팅 옵션 삭제 등 처리 후 저장
  • SSD 인식 불량 해결
    • 파티션은 활성 상태인데 볼륨은 비활성 상태인 경우에 대해 해결됨 확인
    • 자료 : https://kin.naver.com/qna/detail.nhn?d1id=1&dirId=1010305&docId=291566434&qb=7KeI66y4CSDtlZjrk5zrlJTsiqTtgawg7J247Iud7J20IOyViOuQmOuEpOyalA==&enc=utf8§ion=kin&rank=2&search_sort=0&spq=0
      • diskpart list disk select disk 1 # 디스크 번호는 list 결과에 따라 적절히 clean create partition primary # 부팅 디스크로 이용하려면 주(primary) 파티션이어야 함 list partition select partition 1 active # 파티션 활성화 format fs=ntfs quick # 빠른 포맷 진행 exit
