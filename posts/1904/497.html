자료형
    파이썬의 변수는 동적 자료형(실행 도중 변경 가능)으로, 자바와 같은 강한 타입 검사를 하지 않는다. 또, 변수를 나타내는 키워드 없이 식별자만 있으면 변수로써 이용할 수 있다.
    type() 함수를 통해 인자의 자료형을 확인할 수 있다.
  1. bool
    1. True, False
    2. bool 연산 : and, or, not
    3. False로 간주되는 값 : None, 0, 0.0, '', [], (), {}, set()
  2. int
      3. 모든 정수는 int형이며, 크기 제한은 없다.
    1. 정수는 0으로 시작할 수 없다.
    2. 2진수 0b, 8진수 0o, 16진수 0x
    3. / : 부동소수 몫, // : 정수 몫, divmod() : (정수 몫, 나머지)
    4. ** : 거듭제곱
    5. int() : 형변환, 소수점이나 지수가 표현된 문자열은 정수로 변환 불가
  3. float
    1. 부동소수도 +, -, *, /, //, **, %, divmod() 모두 가능하다.
    2. float 정밀도는 그대로 가진다.
    3. float() : 형변환, 소수점이나 지수가 표현된 문자열은 정수로 변환 불가
  4. complex
    5. 복소수. a + bj
  5. str
      6. 파이썬의 문자열은 상수(불변)다. ''와 ""모두 사용 가능하다.
    1. + : 문자열 결합 연산자, * : 문자열 복제 연산자
      문자열 리터럴은 단순히 공백을 기준으로 나열만 해도 연결된다.
    2. ''' 또는 """로 열고 닫음으로써 여러 줄의 문자열 리터럴을 작성할 수 있다.
    3. Raw String 작성
    4. str() : 형변환. str() : 사람이 읽기 편한 형태, repr() : 인터프리터가 인식하는 형태
    5. 슬라이스([start : end : step], end는 미포함)를 통한 문자열 추출
    6. str은 텍스트 시퀀스이므로, len()을 통해 길이를 구할 수 있다.
    7. split() : 분리. 기본은 공백 문자
    8. join() : 문자열 접합
    9. strip() : 앞뒤 문자열 제거. 기본은 공백 제거
    10. 정렬
    11. replace() : 부분 문자열 대체
    12. endswith(), startswith(), find(), rfind(), count(), isalnum()...
    13. capitalize(), title(), upper(), lower(), swapcase(), ...
  6. list
      7. 각 요소를 변경할 수 있는 시퀀스.
    1. 생성 : [], list()
    2. 슬라이스를 통한 리스트 추출
    3. 리스트 병합 : +, extend()
    4. 요소 추가/삭제
    5. 검색
    6. 정렬
    7. 복사
  7. tuple
      8. 각 요소를 변경할 수 없는 시퀀스. 불변하므로 딕셔너리의 키로 사용할 수 있다.
    1. 생성, 병합
    2. 튜플 언패킹
  8. set
    1. 생성, 요소 추가
    2. 원소 포함 여부
    3. 집합 연산
  9. dict
    1. 생성, 병합
    2. 키 포함 여부 : key_name in dict
    3. 키-값 얻기
    4. 복사
구문
  1. 주석 : #
  2. 라인 유지 : \
  3. 조건 분기 : if ~ elif ~ else
  4. while 반복
  5. for 순회
    6. for문은 순회 객체에 대하여 iter()를 호출한다. iter()는 __next__() 메서드를 정의한 iterator 객체를 반환하며, __next__()가 전달하는 값을 하나씩 루프에 사용한다.
    더 이상 요소가 없을 때 __next__()는 StopIteration 예외를 발생시키고, for 루프는 이때 종료한다. 클래스 자체에 __next__()가 정의되어 있다면, iter()는 self를 반환하는 것으로 충분하다.
  6. 반복문 제어 : break, continue
    break 호출 확인 : else
  7. 콤마로 구분된 값들을 자동으로 튜플로 구성하여 다중 할당, 다중 반환이 가능하다.
  8. 순회 대상 생성
    • 수열 생성 : range([start,] stop[, step])
      • start 기본값 0. step 기본값 1. stop 자체는 미포함
    • 여러 시퀀스 동시 순회 : zip()
      • 같은 오프셋의 요소들로 튜플을 생성. 시퀀스가 하나라도 끝나면 순회 종료.
    • comprehension : 순회를 통해 리스트, 딕셔너리, 셋 등 생성
함수
    파이썬의 함수는 일급 객체다.
  1. 함수 정의 def func_name():
  2. return이 없으면 호출자는 None을 얻는다. None 검사 : is None
  3. docstring : 함수 body 첫 줄에 문자열로 함수 설명. help()로 출력된다.
  4. 함수 외부 scope에 접근할 수 있다.
  5. 디폴트 매개변수, 키워드 매개변수, 매개변수 타입 바운딩
  6. * : 인자를 튜플로. ** : 인자를 딕셔너리로.
  7. 함수 내부에 함수 정의 가능
  8. 람다 함수 : lambda
  9. Generator
    10. 자료를 순회할 때마다 다음 항목 반환. __iter__()와 __next__()를 자동으로 생성하고, StopIteration 예외도 자동으로 발생시킨다. cf. range() : 미리 모든 값을 생성한다.
  10. Decorator
    11. 함수 꾸미기. 동적으로 기능 추가.
    * : 인자 리스트 언패킹, ** : 키워드 인자 딕셔너리 언패킹
namespace
    네임스페이스 : "속성 이름 - 객체 이름" 매핑
  1. 메인 프로그램은 전역 네임스페이스를 정의
    2. globals() : 전역 네임스페이스 정보를 담은 딕셔너리
  2. 함수는 지역 네임스페이스를 정의
    3. locals() : 지역 네임스페이스 정보를 담은 딕셔너리
  3. global 키워드로 전역 변수 접근
    4. 출처: Python 3 튜토리얼
