프로세스 관련
  1. ps -ef
    2. ps : process snapshot
    -e : -A와 동일. all processes
    -f : full-format.
    예. ps -ef | grep tomcat
  2. kill -9 pid
    3. 프로세스 종료.
  3. top
    4. 실행 중인 작업 보기.
메모리 관련
  1. vmstat 1
    2. Report virtual memory statistics
  2. free -m -w -h -s 1
    3. Display amount of free and used memory in the system.
    -m : 메가바이트 단위
    -w : wide output
    -h : human-readable output
    -s 1 : 1초마다
시간 동기화
  1. ntpdate == nptd -q
    2. NTP : Network Time Protocol. 시간 동기화를 위한 프로토콜.
    동기화 : ntpdate 서버주소
    시간 비교 : ntpdate -d 서버주소공용 NTP 서버 목록
시각 확인/변경
  1. date
    2. 현재 시각 반환
  2. date -s "time string"
    3. 예. date -s "2019-01-01 12:34:56"
    date --set "time string"과 동일.
