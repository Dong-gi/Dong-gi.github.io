• if문, while문, do-while문, for문, switch문
    • switch 조건으로 변수가 아닌 일반 표현식도 가능
    outerLoop:
for(let i=0; ; ++i) {
    console.log(i);
    if(i > 10)
        break outerLoop;
    else
        continue outerLoop;
}
  • for-in문
    • 객체 프로퍼티 순회
    for(let attr in obj) {}
  • for-of문
    • 배열, Iterable 객체의 각 요소 순회
    for(name of obj) {}
  • 제어문
    • continue, break, return

  • with(obj) : 스코프 고정
  • try ~ catch ~ finally
    • 예외 처리. throw는 임의의 타입을 던질 수 있다.
    try 뒤에 catch 또는 finally가 하나라도 있어야 한다.
    try { throw something; } catch(err) {}
