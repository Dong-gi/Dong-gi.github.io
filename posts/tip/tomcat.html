톰캣 JVM 실행 인코딩/문자셋 설정
  1. catalina.bat
    2. set "JAVA_OPTS=%JAVA_OPTS% -Dfile.encoding=UTF-8 -Dfile.client.encoding=UTF-8 -Dclient.encoding.override=UTF-8"
  2. catalina.sh
    3. JAVA_OPTS="$JAVA_OPTS -Dfile.encoding=UTF-8 -Dfile.client.encoding=UTF-8 -Dclient.encoding.override=UTF-8"
톰캣 리모트 디버깅 설정
  1. catalina.bat
    2. set "JAVA_OPTS=-Xdebug -Xnoagent -Xrunjdwp:transport=dt_socket,address=54433,server=y,suspend=n"
  2. catalina.sh
    3. JAVA_OPTS="-Xdebug -Xnoagent -Xrunjdwp:transport=dt_socket,address=54433,server=y,suspend=n"
  3. Eclipse - Debug Configuration - Remote Java Application
서버 URI 인코딩 설정 : conf/server.xml <Connector port="8080" protocol="HTTP/1.1" connectionTimeout="20000" redirectPort="8443" URIEncoding="UTF-8" />
<Connector port="8009" protocol="AJP/1.3" redirectPort="8443" URIEncoding="UTF-8" />
톰캣 utf8 필터 설정 : {APP_NAME}/web.xml <filter>
　　<filter-name>setCharacterEncodingFilter</filter-name>
　　<filter-class>org.apache.catalina.filters.SetCharacterEncodingFilter</filter-class>
　　<init-param>
　　　　<param-name>encoding</param-name>
　　　　<param-value>UTF-8</param-value>
　　</init-param>
　　<async-supported>true</async-supported>
</filter>

<filter-mapping>
　　<filter-name>setCharacterEncodingFilter</filter-name>
　　<url-pattern>/*</url-pattern>
</filter-mapping>
