이클립스 APT 에러
  1. 예외 : DOMA4011, javax.annotation.processing.FilterException: Source file already created
  2. 원인 : 이클립스 배치 모드에서 특정 파일 수(2000)를 초과하면 애너테이션 처리에 버그 발생
  3. 해결 : eclipse.ini vmargs 섹션에 아래처럼 추가
    4. -vmargs
    -DmaxCompiledUnitsAtOnce=20000
  4. 자료 : https://bugs.eclipse.org/bugs/show_bug.cgi?id=367599
  5. 자료 : https://github.com/palantir/gradle-processors/issues/5
MySQL 시간 기본값 0 설정이 안 될 때
  1. 왜 굳이 now(), NULL, 특정 시각 상수가 아닌 0을 이용하려는지는 모르겠지만...
  2. 원인 : 서버에 NO_ZERO_DATE 옵션이 설정됐기 때문
  3. 옵션 확인 : SHOW VARIABLES LIKE 'sql_mode'
    4. 또는 SELECT @@sql_mode
  4. 해결 1 : SET GLOBAL sql_mode=(SELECT REPLACE(@@sql_mode,'NO_ZERO_DATE',''));
  5. 해결 2 : /etc/my.cnf 수정
    6. [mysqld]
    sql_mode="수정값"
PostgreSQL : remaining connection slots are reserved for non-replication superuser connections
  1. 원인 : 일반적으로 최대 연결 수 부족
  2. 해결 : /var/lib/pgsql/data/postgresql.conf
    3. max_connections=${적당히_큰_숫자}
  3. 자료 : https://stackoverflow.com/a/32584211/7369515
