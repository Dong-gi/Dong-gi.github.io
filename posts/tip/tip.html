윈도우 부트매니저 설정
  • SSD에 설치한 윈도우가 BIOS 설정에서는 검색되지 않는 경우에 대해 해결됨 확인
  • 자료 : https://m.blog.naver.com/pspsim/221141734669
    1. msconfig 실행
    2. 부팅 메뉴에서 기본값 설정, 불필요한 부팅 옵션 삭제 등 처리 후 저장
윈도우 SSD 인식 불량 해결
  • 파티션은 활성 상태인데 볼륨은 비활성 상태인 경우에 대해 해결됨 확인
  • 자료 : https://kin.naver.com/qna/detail.nhn?d1id=1&dirId=1010305&docId=291566434&qb=7KeI66y4CSDtlZjrk5zrlJTsiqTtgawg7J247Iud7J20IOyViOuQmOuEpOyalA==&enc=utf8§ion=kin&rank=2&search_sort=0&spq=0
    1. diskpart
    2. list disk
    3. select disk 1
      4. 디스크 번호는 list 결과에 따라 적절히
    4. clean
    5. create partition primary
      6. 부팅 디스크로 이용하려면 주(primary) 파티션이어야 함
    6. list partition
    7. select partition 1
    8. active
      9. 파티션 활성화
    9. format fs=ntfs quick
      10. 빠른 포맷 진행
    10. exit
