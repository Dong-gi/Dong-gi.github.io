간단 명령어
  1. docker ps
    2. 실행 중인 컨테이너 목록
  2. docker [container] start container_name
    3. 컨테이너 시작
파일 복사
  1. docker-machine scp
    2. docker-machine scp source destination
    예. 로컬 파일을 "kamuvm" 머신으로 복사
    docker-machine scp /root/kamutora/zipang-admin.war kamuvm:/home/docker/
  2. docker cp
    3. docker cp source destination
    예. 로컬 파일을 "devweb" 컨테이너로 복사
    docker cp zipang-admin.war devweb:/usr/local/tomcat/webapps/
