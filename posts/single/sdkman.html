SDKMAN?

SDKMAN은 SDK 관리자로서, 여러 버전의 동일 SDK를 병렬로 설치하고 특정 버전을 골라 사용할 수 있게 해준다.

설치

  1. 자동설치

    2. ↓ shell

    $ curl -s "https://get.sdkman.io" | bash
  2. 스크린 안내대로 나머지 설치 진행

    3. ↓ shell

    $ source "/root/.sdkman/bin/sdkman-init.sh"
  3. 모든 유저가 SDKMAN을 사용할 수 있게 설치
    1. 사용자 디렉터리가 아닌 곳에 설치

      2. ↓ shell

      $ export SDKMAN_DIR="/usr/local/sdkman" && curl -s "https://get.sdkman.io" | bash
    2. sdkman-init.sh 실행에 필요한 권한 부여

      3. ↓ shell

      $ chmod -R +x /usr/local/sdkman/bin/ $ chmod -R +x /usr/local/sdkman/src/
    3. 사용자 로그인 시 자동으로 sdkman-init.sh 실행하도록 설정

      4. ↓ shell

      # .bashrc #... #THIS MUST BE AT THE END OF THE FILE FOR SDKMAN TO WORK!!! cd ~ export SDKMAN_DIR="/usr/local/sdkman" [[ -s "/usr/local/sdkman/bin/sdkman-init.sh" ]] && source "/usr/local/sdkman/bin/sdkman-init.sh" cd ~
