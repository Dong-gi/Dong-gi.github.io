Objects, values, and types
  1. 모든 객체는 id, type, value를 갖는다.
  2. id는 생성된 후에는 불변하며, CPython의 경우 id(obj)는 obj의 메모리 주소다.
  3. type(obj)으로 obj의 타입을 얻을 수 있으며, id와 마찬가지로 불변하다.
  4. Container : 다른 객체에 대한 참조를 포함하는 객체
The standard type hierarchy
  1. None
    2. None 식별자로 이 타입의 싱글톤 객체를 얻을 수 있다. 값이 없음을 알리기 위해 사용된다.
  2. NotImplemented
    3. NotImplemented 식별자로 이 타입의 싱글톤 객체를 얻을 수 있다. 산수 메서드, 비교 메서드는 연산이 구현되지 않은 경우 이를 반환해야 한다.
  3. Ellipsis
    4. 리터럴 ... 이나 Ellipsis 식별자로 이 타입의 싱글톤 객체를 얻을 수 있다. 다음은 NumPy에서 "전체 차원"을 의미하는 ...
  4. numbers.Number; 수
    1. numbers.Integral; 정수
      1. int; 크기 제약 없는 정수
      2. bool; 이며, True, False는 대부분 상황에서 한다.
    2. numbers.Real (float); double precision 부동 소수
    3. numbers.Complex (complex); 실수부, 허수부 모두 float인 복소수.
  5. 시퀀스; 음이 아닌 정수로 인덱싱될 수 있는 유한 길이의 순서 있는 집합
