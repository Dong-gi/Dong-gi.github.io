자료형
  1. 파이썬 변수는 동적 자료형(실행 도중 변경 가능)으로, 실행 중 type() 함수를 통해 인자의 자료형을 확인할 수 있다.
  2. 변수를 나타내는 키워드 없이 식별자만 있으면 변수로써 이용할 수 있다.
    3. bool
    1. True, False
    2. bool 연산 : and, or, not
    3. False로 간주되는 값 : None, 0, 0.0, '', [], (), {}, set()
    int
    1. 모든 정수는 int형이며, 크기 제한은 없다.
    2. 정수는 0으로 시작할 수 없다.
    3. 리터럴 : 2진수 0b, 8진수 0o, 16진수 0x
    4. / : 부동소수 몫, // : 정수 몫, divmod() : (정수 몫, 나머지)
    5. ** : 거듭제곱
    6. int() : 형변환, 소수점이나 지수가 표현된 문자열은 정수로 변환 불가
    float
    1. 부동소수도 +, -, *, /, //, **, %, divmod() 모두 가능하다.
    2. float 정밀도는 그대로 가진다.
    3. float() : 형변환, 소수점이나 지수가 표현된 문자열은 정수로 변환 불가
  3. complex : 복소수 a + bj
    4. str
    1. 여타 언어들과 마찬가지로 파이썬의 문자열은 상수(불변)다. 리터럴로 ''와 ""모두 사용 가능하다.
    2. + : 문자열 결합 연산자
      * : 문자열 복제 연산자
      문자열 리터럴은 단순히 공백을 기준으로 나열만 해도 연결된다.
    3. ''' 또는 """로 열고 닫음으로써 여러 줄의 문자열 리터럴을 작성할 수 있다. 이진 문자열도 가능하다.
    4. 이진 문자열 Raw String : 문자열 리터럴 표기 앞에 r을 붙인다
    5. str() : 사람이 읽기 편한 형태로 객체의 문자열 표현 반환
      repr() : 개발자를 위한 형태로 객체의 문자열 표현 반환. repr() 그대로 객체를 복원할 수 있도록 하자.
    6. 슬라이스([start : end : step], end는 미포함)를 통한 문자열 추출
    7. str은 텍스트 시퀀스이므로, len()을 통해 길이를 구할 수 있다.
    8. split() : 분리. 기본은 공백 문자
      join() : 문자열 접합
    9. strip() : 앞뒤 문자열 제거. 기본은 공백 제거
    10. 정렬
    11. replace() : 부분 문자열 대체
    12. endswith(), startswith(), find(), rfind(), count(), isalnum()...
    13. capitalize(), title(), upper(), lower(), swapcase(), ...
  4. list : 가변 시퀀스
  5. tuple : 불변 시퀀스. 불변이므로 딕셔너리의 키로 사용 가능.
  6. set
  7. dict
구문

  1. for문은 순회 객체에 대하여 iter()를 호출한다. iter()는 __next__() 메서드를 정의한 iterator 객체를 반환하며, __next__()가 전달하는 값을 하나씩 루프에 사용한다. 더 이상 요소가 없을 때 __next__()는 StopIteration 예외를 발생시키고, for 루프는 종료된다. 클래스 자체에 __next__()가 정의되어 있다면, iter()는 self를 반환하는 것으로 충분하다.
  2. 콤마로 구분된 값들을 자동으로 튜플로 구성하여 다중 할당, 다중 반환이 가능하다.
  3. 순회 대상 생성
    • 수열 생성 : range([start,] stop[, step])
      • start 기본값 0. step 기본값 1. stop 자체는 미포함
    • 여러 시퀀스 동시 순회 : zip()
      • 같은 오프셋의 요소들로 튜플을 생성. 시퀀스가 하나라도 끝나면 순회 종료.
    • comprehension : 순회를 통해 리스트, 딕셔너리, 셋 등 생성

  1. 파이썬의 함수는 일급 객체다.
  2. 함수 정의 def func_name():
  3. return이 없으면 호출자는 None을 얻는다. None 검사 : is None
  4. docstring : 함수 body 첫 줄에 문자열로 함수 설명. help()로 출력된다.
  5. 함수 외부 scope에 접근할 수 있다.
  6. 디폴트 매개변수, 키워드 매개변수, 매개변수 타입 바운딩
  7. * : 인자를 튜플로. ** : 인자를 딕셔너리로.
  8. 함수 내부에 함수 정의 가능
  9. 람다 함수 : lambda
  10. Generator
    11. 자료를 순회할 때마다 다음 항목 반환. __iter__()와 __next__()를 자동으로 생성하고, StopIteration 예외도 자동으로 발생시킨다. cf. range() : 미리 모든 값을 생성한다.
  11. Decorator
    12. 함수 꾸미기. 동적으로 기능 추가.
    * : 인자 리스트 언패킹, ** : 키워드 인자 딕셔너리 언패킹

    3. globals() : 전역 네임스페이스 정보를 담은 딕셔너리

  1. 객체 메서드를 표현하는 객체도 속성들을 갖는다.

가상 환경
  1. 서로 다른 프로젝트들이 서로 다른 버전의 패키지 의존성을 가질 수 있다. 이를 유연하게 지원하도록 가상 환경을 만들 수 있다.
  2. 최신 파이썬 버전에는 venv 모듈이 포함되어 있다.
  3. python3 -m venv my-env1
    4. my-env1 디렉터리가 존재하지 않으면 생성한다. 이 디렉터리에 파이썬 인터프리터, 표준 라이브러리 등이 복사된다.
  4. Windows에서 실행 : my-env1\Scripts\activate.bat
  5. Unix에서 실행 : source my-env1/bin/activate
