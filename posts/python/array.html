class array.array
  • class array.array(typecode[, initializer])
    • initializer : list, bytes-like object, iterable 중 하나를 보내면 fromlist(), frombytes() 등으로 초기화된다
  • 모듈 속성
    1. array.typecodes
      2. 유효 타입 코드들로 구성된 문자열
  • 인스턴스 속성(읽기 전용)
    1. array.typecode
    2. array.itemsize : 요소 하나의 바이트 크기
  • 인스턴스 메서드
    1. array.buffer_info()
      2. (address, length) 반환. 반환된 값은 배열이 삭제되지 않고, 길이가 변경되지 않은 동안 유효
    2. array.byteswap()
      3. 배열 각 요소들의 바이트 순서를 뒤집는다. array.itemsize가 1, 2, 4, 8인 경우만 작동
    3. array.extend(iterable)
      4. 배열 끝에 순서대로 추가. 인자가 배열인 경우, 타입이 동일해야만 작동한다
    4. array.fromfile(f, n) ↔ tofile(f)
      5. file 인스턴스 f로부터 n개의 machine value를 읽어 배열의 끝에 순서대로 추가. 요소가 n개보다 적은 경우 배열에 추가는 되지만 EOFError 발생
    5. array.frombytes(s) Since 3.2 ↔ tobytes()
      6. 문자열을 machine value로 읽어 배열 끝에 순서대로 추가
    6. array.fromlist(list) ↔ tolist()
      7. list 전체 요소의 타입이 삽입 가능하다면, 순서대로 배열 끝에 추가
    7. array.index(x)
    8. array.count(x)
    9. array.remove(x)
    10. array.insert(i, x)
    11. array.pop([i])
    12. array.reverse()
