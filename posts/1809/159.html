1. 필드
@Native static final int TCP_NODELAY : Nagle 알고리즘 작동 중지
@Native static final int SO_BINDADDR : Local 주소 바인딩. SocketImpl, DatagramSocketImpl의 생성자에서 설정해야 한다.
@Native static final int SO_REUSEADDR : MulticastSocket에서만 사용되며, 기본 설정된다.
@Native static final int SO_REUSEPORT Since 9 : 여러 소켓이 같은 주소와 포트를 listen.
@Native static final int SO_BROADCAST Since 1.4 : 프로세스가 브로드캐스트 메시지를 보낼 수 있는지. 브로드캐스트를 지원하는 네트워크에서, 데이터그램 소켓을 통해서만 지원된다. 데이터그램 소켓은 이를 기본으로 설정한다.
@Native static final int IP_MULTICAST_IF : 밖으로 나가는 인터페이스가 멀티캐스트 패킷을 보내는지 여부. 호스트가 여러 네트워크 인터페이스를 가진 경우, 시스템 기본이 아닌 다른 것을 이용할 때 InetAddress를 인자로 설정. DatagramSocketImpl에서 유효.
@Native static final int IP_MULTICAST_IF2 Since 1.4 : IPv6 지원.
@Native static final int IP_MULTICAST_LOOP Since 1.4 : 멀티캐스트 데이터그램 루프백 여부. MulticastSocket은 기본으로 설정.
@Native static final int IP_TOS Since 1.4 : TCP/UDP 소켓의 IP 헤더에 type-of-service 또는 traffic class field 설정
@Native static final int SO_LINGER : TCP 소켓의 close()와 관련. linger-on-close timeout을 설정하여 close()가 즉시 리턴하도록 할 수 있다.
@Native static final int SO_TIMEOUT : ServerSocket::accept, SocketInputStream::read, DatagramSocket::receive에 대한 timeout 설정
@Native static final int SO_SNDBUF : 전송 버퍼 크기에 대한 힌트
@Native static final int SO_RCVBUF : 수신 버퍼 크기에 대한 힌트
@Native static final int SO_KEEPALIVE : 2시간 동안 아무 교환이 없으면 https://docs.oracle.com/javase/10/docs/api/java/net/SocketOptions.html#SO_KEEPALIVE의 동작을 수행
@Native static final int SO_OOBINLINE : 설정되면 input stream을 통해 긴급 데이터(1바이트)를 받을 수 있다.
