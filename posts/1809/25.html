• Java 1.8 부터 버전 표기를 싱글 넘버로 표기하기 시작.
  • 람다 표현식 추가
  • Pipeline과 Stream 추가
  • 인터페이스에 Default Method 추가
  • Type Annotation 추가. https://checkerframework.org/
    • @NonNull : 컴파일러가 null이 대입되는지 확인
    • @ReadOnly : 컴파일러가 쓰기 작업이 있는지 확인
    • @Regex : 컴파일러가 정규표현식 검증
    • @Untainted : 오염되지 않은(@Untainted) 값임을 명시. 또는 그러한 값만을 받아들임을 명시
    • @Tainted : 값이 오염된(@Tainted) 가능성이 있음을 명시. 또는 오염된 값도 받아들임을 명시
    • @m : 미터, 초 등의 단위 체크
  • Collection API 강화
