1. JavaEE
    2. Java Enterprise Edition은 동적 웹 애플리케이션 개발을 위한 자바 플랫폼이다.
    JavaSE와 다른 문서 페이지를 참조해야 한다. https://javaee.github.io/javaee-spec/javadocs/
    WAR(Web Application Archive) : JavaEE 웹 애플리케이션을 위한 압축 파일

    웹 애플리케이션 루트
    ㄴ META-INF
    ㄴ MANIFEST.MF
    ㄴ 컨테이너 지원
    ㄴ WEB-INF : 웹에서 접근 불가
    ㄴ classes
    ㄴ META-INF
    ㄴ 애플리케이션 자원
    ㄴ 자바 .class 파일 및 자원
    ㄴ i88n
    ㄴ 국제화 파일
    ㄴ lib
    ㄴ 번들 jar 파일
    ㄴ tags
    ㄴ JSP 태그 파일
    ㄴ tld
    ㄴ JSP 태그 라이브러리 설명자
    ㄴ 기타 웹에서 접근 가능한 파일
