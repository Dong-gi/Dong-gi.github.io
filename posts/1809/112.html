  2. static final class Runtime.Version implements Comparable<Runtime.Version> Since 9
    • $VNUM : [1-9][0-9]*((\.0)*\.[1-9][0-9]*)*
    • 보통 다음과 같은 의미를 나타낸다. $FEATURE.$INTERIM.$UPDATE.$PATCH
    • $FEATURE : Feature-release counter, $INTERIM : Interim-release counter, $UPDATE : Update-release counter, $PATCH : Emergency patch-release counter
    • Version String : $VNUM(-$PRE)?\+$BUILD(-$OPT)? or $VNUM-$PRE(-$OPT)? or $VNUM(+-$OPT)?
    • $PRE : [a-zA-Z0-9]+ : Pre-release identifier, $BUILD : 0|[1-9][0-9]* : Build number, $OPT : [-a-zA-Z0-9.]+ : Additional build information
