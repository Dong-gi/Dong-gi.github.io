• 기본 자료형
    • 1. 정수 char(1byte), short(2bytes), int(short <= int <= long), long(4bytes) // unsigned 키워드로 non-negative 변수로 이용 가능.
    2. 부동소수 float(4bytes), double(8bytes), long double(10bytes)
    3. void : 반환 타입 선언 | 매개변수 선언 | void 포인터
  • 리터럴
    • 1. unsigned : 123U, long : 123L, 16진수 : 0xFFFF, 8진수 : 07777
    2. 지수 표기 : 1.23E, float : 1.F
  • 상수
    • 1. const float PI = 3.141592F
    2. #define PI 3.141592 // 컴파일러가 단순 치환 작업함
  • 메모리에 따른 변수 종류
    • 1. auto : 모든 지역변수 기본값("int a;"와 "auto int a;"는 같다). 블록이 끝나면 소멸
    2. register : RAM이 아닌 레지스터에 변수 생성.
    3. static : 처음 호출 시 한번만 초기화되고 프로그램 종료시까지 유지
    4. extern : 외부 파일 전역 변수 참조
  • 연산자
    • 1. 산술 : +, -, *, /, %, ++, --
    2. 관계 : ==, !=, >, >=, <, <=
    3. 논리 : &&, ||, !
    4. 비트 : &, |, ^(xor), ~(not), <<, >>

    5. sizeof(), &(address), *(pointer), ?:(condition), (형변환), [], ->
  • 구문
    • 1. if ~ else if ~ else // 0만 아니면 다 true

    2. switch : 정수 타입만 가능

    3. while, for, do ~ while
  • 매개변수 전달
    • 1. call by value : 값의 복사에 의한 인자 전달

    2. call by reference : 주소 복사에 의한 인자 전달
  • 변수
    • 1. 지역변수는 자동 초기화되지 않는다.

    2. 전역변수 초기화값 : 0, '\0', 0.f, 0., NULL
  • 배열
    • 1. type 식별자[size];

    2. int arr[] = {1, 2, 3};

    3. int arr[3][3];

    4. 배열 이름은 그 시작 주소를 가리키는 포인터다.
  • 포인터
    • 1. 포인터에 대한 ++, -- 연산은 sizeof()값만큼의 주소 이동이다.

    2. int *arr[3]; // 포인터의 배열

    3. int (*arr)[]; // 배열의 포인터
  • 문자열
    • 1. 문자열은 '\0'으로 끝난다.

    2. char str[] = "Hello World";

    3. char *str = "Hello World";

    4. string.h :: strcpy(), strcat(), strlen(), strcmp(), strchr() : index of a char, strstr() : index of a string
  • 구조체
    • 1. struct [식별자] { 멤버변수 목록 }

    2. 구조체 멤버 접근은 . 또는 ->(포인터)로 이루어진다.

    3. 비트필드 : 각 멤버의 비트 수를 제한할 수 있다.
    » struct { type 식별자 : 비트수; }
  • typedef
    • 1. 기정의된 타입을 다른 이름으로 이용할 수 있게한다.
    » typedef unsigned int UINT_32
  • 전처리기
    • 1. 전처리기 : #include, #define, #undef, #ifdef, #ifndef, #if, #elif, #else, #endif, #pragma

    2. 매크로 : __DATE__, __TIME__, __FILE__, __LINE__

    3. 매크로 : #define name(params) (처리)

    4. #(stringize), ##(token pasting) : #define STRCAT(str1, str2) \
    printf("str1 : " #str1 ", str2 : " #str2 ", result : %s", str1##str2)
