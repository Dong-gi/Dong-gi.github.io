• 식별자 : '_', '$' 허용.
  • 변수 선언
    • - 자바스크립트의 변수는 타입 체크 대상이 아니다.
    - 하지만 typeof 연산자로 타입을 가져와 이용할 수는 있다.
    • let 식별자;
      • - 일반적인 지역변수와 동일. 선언된 순간부터 해당 코드 블록이 끝날때까지 유효.
      - 중첩된 코드 블록에서 같은 식별자의 변수를 선언하여 밖의 변수를 가릴 수 있다.
    • var 식별자;
      • - 선언 전에 참조하는 경우, 선언을 끌어올려 undefined를 할당해준다.
      - 함수 스코프에서 동작하지만, 중첩된 코드 블록에서 재정의하여 가릴 수 없다.
  • 상수 선언 : const 식별자;
  • 타입
    • - Object, String, Number, Boolean, Undefined, Null, Symbol
    » Boolean() : "", 0, NaN, null, undefined == false, 그 외 == true
    따라서 (null == undefined) === true
    » Number(), parseInt(), parseFloat(), number.toString(radix)
    » 문자 리터럴 : \xFF, \uFFFF
    » 문자열 템플릿 ``
    » Symbol
    ES6부터 도입된 타입. 그 무엇과도 같지 않은 유일한 토큰을 나타낸다.
    const SYMBOL = Symbol();
    » Object : {}
    - 객체는 SIZE 심볼을 갖는다.
    - delete 키워드로 프로퍼티 제거 가능.
  • 배열 : []
    • - 자바스크립트는 모두 동적 배열. 또, 변수와 마찬가지로 타입을 가리지 않는다. - length 프로퍼티에 개수 저장.
