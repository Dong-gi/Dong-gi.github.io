1. if문, while문, do-while문, for문, switch문
- switch 조건으로 변수가 아닌 일반 표현식도 가능
outerLoop:
for(let i=0; ; ++i) {
    console.log(i);
    if(i > 10)
        break outerLoop;
    else
        continue outerLoop;
}

2. for-in문
- 객체 프로퍼티 순회
for(let attr in obj) {}

3. for-of문
- 배열, Iterable 객체의 각 요소 순회
for(name of obj) {}

4. 제어문
- continue, break, return

5. with(obj) : 스코프 고정
