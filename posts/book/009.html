글귀

  • Availability heuristic

    • 쉽게 떠오르는 것이 더 발생할 확률이 높다고 판단

  • Conjunction fallacy

    • 두 사건의 결합 확률(더 자세하고 그럴싸한 설명)을 각각의 발생 확률보다 크다고 판단

  • Outcome bias

    • 복잡한 과정은 무시하고 알기 쉬운 결과를 통해 판단

  • Base rate neglect

    • 기저(사전 확률)를 무시하고 결과에 시간적으로 가까운 사후 확률로 판단

  • Anchoring heuristic

    • 최초 제시된 정보에 더 큰 가중치를 두고 판단

  • Mere exposure effect

    • 자주 보면 호감도가 오른다

  • Mere thought effect

    • 자주 생각할수록 중요하다고 여긴다. 물론 과유불급

  • Backfire effect

    • 믿음에 반대되는 사실을 접하면 오히려 믿음을 강화

  • Endowment effect

    • 자기 소유라는 사실 만으로 더 가치있다고 판단

  • Loss aversion

    • 같은 양의 손익에 대해, 손실에 더 민감하게 반응

  • Status quo bias

    • 현상 유지 편향

  • Negativity effect

    • 타인을 평가할 때는 부정적인 정보에 큰 가중치를

  • Restraint bias

    • 자신의 절제력을 과대평가

  • Optimism bias

    • 근거 없이 미래를 낙관

  • Bandwagon effect

    • 다수의 의견(가짜든 아니든)을 따라감. 삼인성호

  • Wishful thinking bias

    • 자기에게 긍정적인 것은 쉽게 믿음

  • Ben franklin effect

    • 내가 호의를 베푼 대상을 더 긍정적으로 평가. 내가 상처를 준 대상은 더 부정적으로 평가

  • Zeigarnik effect

    • 미완성 과제에 더 집착
