  • 대조군이 없으면 비교할 수 없습니다

    같은 그룹 안에서 무작위로 구성원을 선택해 실험군과 대조군을 구성하면, 둘은 비슷한 수준으로 통제변인의 영향을 받으므로 조작변인에 의한 비교가 가능하다

  • 실험군과 같은 시기를 체험하는 동시 대조와 달리, 과거 데이터를 대조군으로 선택하는 경우 실험군과 통제변인이 다르기 쉽다. 때문에 어떤 것을 시험하든 실험군의 성공을 지지하는 경향이 있으며 -- 이러한 실험은 의심해봐야 한다
  • 산포도 : x축에 독립 변수, y축에 종속 변수를 설정하고 x-y 사이의 상관 관계를 찾는다. 물론 상관 관계가 존재한다고 그것이 인과 관계를 증명하는 것은 아니다(필요조건)
  • 현실은 다변량 데이터이며, 인과관계는 선형이 아니라 네트워크 형이다
  • 여러 가설 중 가장 마음에 드는 하나를 선택하는 것보다, 말이 되지 않는 가설을 하나씩 지워나가는 반증이 낫다
  • 베이즈 정리의 이용 - 기준 비율 오류에 주의!

    ↓ text

    통계 결과, 전 국민의 1%가 신종 독감에 감염되었음이 밝혀졌습니다 -- 기준 비율 1% 와중에, 당신은 신속항원검사에서 양성 판정이 나왔습니다. 실제로 양성일 확률은 어떻게 될까요? - 신속항원검사 : 독감에 걸렸을 경우 양성 판정 확률 90%, 걸리지 않았을 경우 양성 판정 확률 9% P(실제 양성|검사 양성) = P(검사 양성 & 실제 양성) / P(검사 양성) = P(실제 양성) * P(검사 양성|실제 양성) / (P(실제 양성) * P(검사 양성|실제 양성) + P(실제 음성) * P(검사 양성|실제 음성)) = 0.01 * 0.9 / (0.01 * 0.9 + 0.99 * 0.09) = 약 9% 9%? 1%보다 높긴 한데... 독감에 걸렸다고 말할 수 있는 수치는 아닌 것 같네요. 조금 비싸지만 보다 정확한 검사를 받아보았더니 음성 판정이 나왔습니다. 실제로 양성일 확률은 어떻게 될까요? - 고급항원검사 : 독감에 걸렸을 경우 양성 판정 확률 99%, 걸리지 않았을 경우 양성 판정 확률 1% 잠깐! 당신은 이미 신속항원검사를 받았기 때문에, 당신의 기준 양성 확률은 이제 1%가 아니라, 9%입니다. P(실제 양성|검사 음성) = P(검사 음성 & 실제 양성) / P(검사 음성) = P(실제 양성) * P(검사 음성|실제 양성) / (P(실제 양성) * P(검사 음성|실제 양성) + P(실제 음성) * P(검사 음성|실제 음성)) = 0.09 * 0.01 / (0.09 * 0.01 + 0.91 * 0.99) = 약 0.1% 안심하고 결과를 받아들일 수 있겠네요.
  • 베이즈 정리의 이용 - 새로운 증거가 나왔을 때, 주관적 확률을 점진적으로 수정하는 방법

    ↓ text

    러시아가 다음 분기에 석유 산업에 보조금을 지원할 주관적 확률을 90%라고 생각했을 때, 러시아가 유전을 매각할 것이라는 뉴스가 보도되었습니다. - 물론 뉴스가 그대로 현실이 되리란 법은 없습니다 하지만 러시아 석유 산업에 대한 투자를 유지할 지, 매각할 지 등의 결정을 위해 새롭게 판단할 필요가 있네요 이때, 단순히 새로운 주관적 확률을 제시하는 것보다 베이즈 정리를 이용하는 것이 오류를 줄이는 보다 정확한 방법입니다! 우리가 알고 싶은 것은 러시아가 다음 분기에 보조금을 계속 지원할 것인가?하는 것입니다 P(지원금O | 매각 발표O) = P(지원금O) * P(매각 발표O | 지원금O) / (P(지원금O) * P(매각 발표O | 지원금O) + P(지원금X) * P(매각 발표O | 지원금X)) = 0.9 * P(매각 발표O | 지원금O) / (0.9 * P(매각 발표O | 지원금O) + 0.1 * P(매각 발표O | 지원금X)) 이제 여러분은 다음 두 가지에 대한 주관적 확률을 구하면 새로운 판단을 할 수 있습니다! - P(매각 발표O | 지원금O) : 지원금을 지급하면서 유전 매각 뉴스를 발표할 확률 -- 러시아의 블러핑! - P(매각 발표O | 지원금X) : 지원금을 지급하지 않으면서 유전 매각 뉴스를 발표할 확률 -- 뉴스의 현실화!
