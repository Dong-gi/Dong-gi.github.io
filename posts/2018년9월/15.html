1. C 네이티브 코드를 이용할 클래스 컴파일 : javac Test.java -> Test.class 생성
  2. C에서 이용할 헤더 파일 생성 : javah -jni Test -> Test.h 생성
  3. VS에서 JDK path 추가 : 속성 - 포함 디렉터리 - "$(JAVA_HOME)\include", "$(JAVA_HOME)\include\win32 추가
  4. VS에서 dll 빌드 : 속성 - 구성(출력) 형식 - 동적 라이브러리(.dll) 설정
  5. 빌드한 dll을 이클립스 프로젝트 폴더 안으로 이동. x86 JRE/JDK를 이용하여 실행(Window - Preference - Installed JREs에 x86 있는지 확인. Run - Run Configurations... - x86 JRE로 설정)
