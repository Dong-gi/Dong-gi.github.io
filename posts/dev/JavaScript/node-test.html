Test runner

커맨드 라인을 통해 테스트를 실행할 수 있다

↓bash

node --test

이때 실행되는 파일들은 아래와 같다

  • **/*.test.?(c|m)js
  • **/*-test.?(c|m)js
  • **/*_test.?(c|m)js
  • **/test-*.?(c|m)js
  • **/test.?(c|m)js
  • **/test/**/*.?(c|m)js

또는 직접 GLOB 패턴을 지정할 수도 있다

↓bash

node --test "**/*.spec.js"
