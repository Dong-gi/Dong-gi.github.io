이벤트

  1. 이벤트 강제 발생

    2. ↓ javascript

    $(node).trigger('eventName'); $.event.trigger(); // 사용자 정의 이벤트 강제 발생
  2. 이벤트 핸들러 추가/제거

    1. ↓ javascript

      .on(events[, selector][, data], handler)

      2. selector가 null이거나 제공되지 않은 경우, 선택된 노드들에 대해 이벤트 핸들러 추가. selector가 제공된 경우 선택자에 매칭되는 후손 노드(미래에 추가되는 것들도 포함)들에 핸들러가 연결된다

    2. ↓ javascript

      .one(events[, selector][, data], handler)

      3. 핸들러가 한 번만 이벤트를 처리한다
