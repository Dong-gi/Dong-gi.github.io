브라우저와 관련된 객체 : window, screen, location, history, navigator, document

    window : 브라우저 윈도우를 가리키는 객체
    screen : window.screen과 동일. 표시되는 화면과 연관
    location : window.location과 동일. 현재 페이지 URL과 연관
    history : window.history와 동일. 페이지 기록과 연관
    navigator : window.navigator와 동일. 브라우저 자체와 연관
    document : window.document와 동일. 표시되는 페이지 내용과 연관
