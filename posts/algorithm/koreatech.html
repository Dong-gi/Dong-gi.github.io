1003 : 0을 만들자 - Small
  1. 길이 n인 정수 순열이 주어 졌을 때, 그 안에 숫자를 3개 골라서 합이 0(Zero) 이 되는 조합이 몇 개 있는지 출력하는 프로그램을 만들어 주세요.
  2. (a, b, c)의 조합은 순서와 상관이 없으므로, a <= b <= c인 조합만 고려하면 된다. 이를 (small, center, big)으로 표현하면, big = -(small+center)가 성립한다.
    1. small < center < big인 경우 : 전체 수열에서 big이 적어도 한번 등장하는지 체크한다. (-1, 0, 1)과 같은 경우.
    2. small < center = big인 경우 : 전체 수열에서 big이 적어도 두번 등장하는지 체크한다. (-2, 1, 1)과 같은 경우.
    3. small = center = big인 경우 : 전체 수열에서 big이 적어도 세번 등장하는지 체크한다. (0, 0, 0)의 경우
    4. small = center < big인 경우 : 전체 수열에서 big이 적어도 한번 등장하는지 체크한다. (-1, -1, 2)와 같은 경우.

1019 : 숫자 바꿔치기
  1. 김교수와 하사장은 숫자 자릿수가 변하는 것은 기가 막히게 알아채지만, 자릿수가 같을 때는 두 숫자의 위치가 바뀌는 것은 잘 알아채지를 못한다. 이를 이용하여, 김교수에게는 단가를 크게 만들어서 많은 돈을 받고, 하사장에게는 단가를 작게 만들어서 적은 돈을 주고 남은 차액은 본인이 챙긴다. 단가가 주어질 때 영준이의 부당이익을 계산하는 프로그램을 작성하라.
  2. 일단 구성하는 모든 자리수를 따로 정렬한다.
  3. 큰 수를 만드는 것은 간단 : 큰 수부터 검사하여 크기가 다르면 교체
  4. 작은 수의 경우, 0이 있는 경우 : 0이 아닌 다른 가장 작은 수가 있으면 교체
  5. 작은 수의 경우, 0이 없는 경우 : 큰 수와 비슷하게, 작은 수부터 검사하여 크기가 다르면 교체
