JSON
  • public 필드 또는 public setter가 존재하는 POJO
  • 제네릭 List, Map도 가능.
  • JSON 노드 수정.
  • 직렬화/역직렬화 관련 옵션
    • DateFormat 설정 : GsonBuilder#setDateFormat
    • 직렬화 제외 설정 : GsonBuilder#excludeFieldsWithModifiers
      • @Expose with GsonBuilder#excludeFieldsWithoutExposeAnnotation
    • 직렬화 시 정렬 : GsonBuilder#setPrettyPrinting
    • Null 허용 : GsonBuilder#serializeNulls
    • 사용자 정의 어댑터 : GsonBuilder#registerTypeAdapter
      • TypeAdapter, InstanceCreator, JsonSerializer, JsonDeserializer 중 하나 이상의 조합
    • 속성명 임의 설정 : @SerializedName("custom_naming")
Android ProGuard 관련 설정
\n##---------------Begin: proguard configuration for Gson  ----------
# 출처 : https://github.com/google/gson/blob/master/examples/android-proguard-example/proguard.cfg
# Gson uses generic type information stored in a class file when working with fields. Proguard
# removes such information by default, so configure it to keep all of it.
-keepattributes Signature
# For using GSON @Expose annotation
-keepattributes *Annotation*
# Gson specific classes
-dontwarn sun.misc.**
#-keep class com.google.gson.stream.** { *; }
# Application classes that will be serialized/deserialized over Gson
-keep class com.google.gson.examples.android.model.** {
