주요 인터페이스

Channel

I/O 작업을 지원하는 네트워크 연결. 모든 I/O 작업은 비동기적으로 실행되어 ChannelFuture를 즉시 반환한다

ChannelFuture

  • Channel의 I/O 작업 결과
  • 가능한 상태 : Uncompleted(!isDone), Success(isDone && isSuccess), Failed(isDone && cause != null), Cacelled(isDone && isCancelled)
  • addListener(s) 메서드를 이용해 콜백 설정 가능
