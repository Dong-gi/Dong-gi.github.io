JSON
  • public 필드 또는 public setter가 존재하는 POJO
  • 제네릭 List, Map도 가능.
  • JSON 노드 수정.
  • JSON 직렬화/역직렬화에 직접 관여할 수 있다.
  • 직렬화/역직렬화 관련 옵션
    • https://github.com/FasterXML/jackson-databind/wiki/JacksonFeatures
    • https://github.com/FasterXML/jackson-docs/wiki/JacksonAnnotations
    • DateFormat 설정 : mapper.setDateFormat(DateFormat)
    • 직렬화 제외 설정 : mapper.setSerializationInclusion(JsonInclude.Include);
    • 직렬화 시 정렬 : mapper.enable(SerializationFeature.INDENT_OUTPUT);
    • 빈 객체 직렬화 허용 : mapper.disable(SerializationFeature.FAIL_ON_EMPTY_BEANS);
    • java.util.Date, Calendar를 timestamp 숫자로 쓰지 않기 : mapper.disable(SerializationFeature.WRITE_DATES_AS_TIMESTAMPS);
    • 매칭되지 않은 속성은 건너뛰기 : mapper.disable(DeserializationFeature.FAIL_ON_UNKNOWN_PROPERTIES);
    • 속성명을 ""로 감싸지 않아도 됨 : mapper.enable(JsonParser.Feature.ALLOW_UNQUOTED_FIELD_NAMES);
    • ""대신 '' 허용 : mapper.enable(JsonParser.Feature.ALLOW_SINGLE_QUOTES);
    • Non-ASCII 이스케이프 : JsonMapper.Builder#configure(JsonWriteFeature.ESCAPE_NON_ASCII, true)
    • 애너테이션 기반 설정 : @JsonProperty("newName"), @JsonIgnoreProperties({ "foo", "bar" }), @JsonCreator, @JsonSetter, @JsonGetter, ...
