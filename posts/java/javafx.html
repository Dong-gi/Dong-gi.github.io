  4. 노드
    • JavaFX UI는 노드의 계층적인 구성으로 이루어진다.
    • 컨테이너 노드 : 다른 노드를 내부에 포함. Stage, Scene, Pane...
    • 기본 노드 : Button, Label, Checkbox...
    • Layout Manager : BorderPane, FlowPane, StackPane, HBox, VBox...
    • Stage는 윈도우 창으로, 한 번에 하나의 Scene을 보여준다. 하나의 Scene은 하나 이상의 Pane으로 구성된다.
    • Modal : 해당 윈도우가 종료될 때까지 부모 윈도우와 상호작용 불가. ↔ modaless
    • 그리기 : javafx.scene.shape
