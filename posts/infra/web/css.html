Box Model

CSS가 표시하는 모든 것은 박스다

각 박스는 하위 영역들(다시 박스)로 구성된다; '컨텐츠 박스' > '패딩 박스' > '테두리 박스' > '여백 박스'

  • 컨텐츠 박스 : 자식 요소들이 포함될 수 있고, 컨텐츠 박스의 크기가 자식 요소들의 크기에 의해 (동적으로) 결정되도록 할 수도 있다
  • 패딩 박스 : 컨텐츠 박스를 감싸는 영역으로, 요소에 적용된 배경은 이 영역까지 표시된다. 또, 오버플로 속성으로 인한 스크롤바도 이 영역에 표시된다
  • 테두리 박스 : 패딩 박스를 감싸는 영역으로, 박스의 경계를 표시한다. border 속성으로 제어한다.
  • 여백 박스 : 박스 바깥의 공간으로, 여백의 크기는 박스의 크기에 영향을 미치지 않는다.

박스 모델 제어

  • 모든 웹 브라우저는 HTML 문서에 기본 스타일시트를 적용한다
  • display 속성 : 요소를 block 또는 inline 요소를 취급하고, 자식 박스들을 어떻게 배치할 지 제어한다
  • box-sizing 속성 : width, height를 컨텐츠 박스까지 적용할 지(기본값), 테두리 박스까지 적용할 지 제어한다

선택자

단순 선택자

  • 범용 선택자

    • * : 모든 요소를 선택한다

  • 타입 선택자

    • div : 모든 div 요소를 선택한다

  • 클래스 선택자

    • .dialog : dialog 클래스를 가진 모든 요소를 선택한다

  • ID 선택자

    • #target : target ID를 가진 요소를 선택한다

  • 속성 선택자
    • [attr] : attr 속성을 갖는 요소
    • [attr='value'] : attr 속성값이 'value'와 일치하는 요소
    • [attr='value' s] : attr 속성값이 'value'와 일치하는 요소(대소문자 구별 O)
    • [attr='value' i] : attr 속성값이 'value'와 일치하는 요소(대소문자 구별 X)
    • [attr*='value'] : attr 속성값이 'value'를 포함하는 요소
    • [attr^='value'] : attr 속성값이 'value'로 시작하는 요소
    • [attr$='value'] : attr 속성값이 'value'로 끝나는 요소
  • 그룹화 선택자

    • 콤마(,)를 이용해 같은 스타일을 여러 요소에 적용시킬 수 있다

    ↓css

    /* strong 요소, em 요소, lang 속성을 가진 요소의 글자 색을 빨강으로 한다 */ strong, em, [lang] { color: red; }

의사 클래스 및 의사 요소

의사 클래스

  • 요소들의 상태를 기준으로 선택하는 데 사용할 수 있는 의사 클래스가 정의되어 있다. ':'을 이용해 의사 클래스를 지정한다
  • a: active, hover, link, visited
  • input: checked, disabled, enabled, focus, in-range, out-of-range, valid, invalid, read-only, read-write, required, optional
  • :first-child, last-child, nth-child(n), nth-last-child(n)
  • :not(selector)

의사 요소

  • 요소의 특정 부분을 선택하거나, 요소의 앞뒤에 내용을 추가하는 데 사용한다. '::'를 이용해 의사 요소를 지정한다
  • ::first-letter : 첫 번째 글자
  • ::first-line : 첫 번째 줄
  • ::marker : 리스트 아이템의 글머리 기호(또는 번호)
  • ::before : 요소 앞부분에 추가
  • ::after : 요소 뒷부분에 추가
  • ::selection : 블록 지정된 요소

결합기(Combinator)

결합기는 두 선택자 사이에 위치하여, 특정한 관계를 만족하는 요소를 선택할 수 있게 해준다

  • '>' : 자식 요소
  • ' '(스페이스) : 후손 요소
  • '+' : 바로 다음 형제 요소
  • '~' : 다음 모든 형제 요소

The cascade

요소에 대해 둘 이상의 스타일이 존재할 때, 브라우저는 아래 순서에 따라 적용할 스타일을 결정한다

스타일의 위치와 순서

  • 동일한 CSS 안에서의 충돌 : 스크립트 뒷부분에 정의된 것이 우선
  • 두 CSS 간 충돌 : 보다 나중에 로드/정의된 것이 우선
  • 변수
    1. 변수 선언

      2. ↓css

      root { --var1: 12px; --my-var2: blue; }
    2. 변수 사용

      3. ↓css

      p { color: var(--my-var2); }
  • meadia query

    • Syntax::= @media not|only mediatype and (media feature and|or|not mediafeature) { CSS-Code; }css3_pr_mediaquery

    디자인

    • 공통 속성
      1. 배경색 background-color: color;
      2. 배경 이미지 background-image: url("url")

        3. 이미지 잘라넣기 : 투명한 img로 자리를 차지하고, background: url('url') 시작w 시작h; 설정

        외곽 이미지배경 이미지이미지 스타일링img 크기에 맞추는 방식

      3. 외곽선 border-style: dotted | dashed | solid | double | groove | none | hidden | ...
        • 두께 border-width: width; » 색상 border-color: color; » 곡률 border-radius: radius;
        • 방향별로 설정 가능 : top, bottom, left, right
        • margin 밖의 외곽선 outline도 존재
      4. 외부 여백 margin: auto | size | % | inherit;
        내부 여백 padding: size | % | inherit;
      5. 너비 width: auto | size | %, 높이 height: auto | size | %
        • 최대 너비 max-width: none | size | %, 최대 높이 max-height: none | size | %
        • 최소 너비 min-width: size | %, 최소 높이 min-height: size | %
    • 텍스트
      1. 색상 color: color;
      2. 정렬 text-align: left | center | right | justify(양쪽정렬)
      3. 꾸밈선 text-decoration: none | overline | line-through | underline
      4. 대소문자 강제 text-transform: uppercase | lowercase | capitalize
      5. 들여쓰기 text-indent: size
      6. 자간 letter-spacing: size, 단어 간격 word-spacing: size
      7. 글자 크기 font-size: size, 줄 간격을 포함한 높이 line-height: size(배율)
      8. 글자 두께 font-weight: normal|bold|bolder|lighter|number|initial|inherit;
        그림자 text-shadow: h-shadow v-shadow blur-radius color|none|initial|inherit;
      9. 폰트 font-family: family-name|generic-family|initial|inherit;
      10. 영역을 벗어난 텍스트 처리 text-overflow, text-break, text-wrap참고
    • 리스트
      1. 리스트 아이템 이미지 마크 설정 list-style-image: none|url|initial|inherit;
      2. 리스트 아이템 마커 타입 list-style-type: value;참고
      3. 리스트 아이템 마커 위치 list-style-position: inside | outside;
    • 보여주기 속성
      1. display: none | block | inline | inline-block(너비/높이 속성을 가질 수 있다)참고
      2. visibility: visible | hidden(자리는 차지) | collapse | initial | inherit;
      3. 위치 position: static | relative | fixed | absolute | sticky;
      4. z축 우선순위 z-index: number;
      5. 넘침 영역 overflow, overflow-x, overflow-y: visible | hidden | scroll | auto | initial | inherit;
      6. 띄우기 float: none | left | right | initial | inherit;
      7. float 요소를 배제하여 구분된 영역 이용 clear: none | left | right | both | inherit;
      8. 불투명도 opacity: [0.0-1.0]; 또는 RGBA 이용
      9. box-shadow: none|h-offset v-offset blur spread color |inset|initial|inherit;
      10. 다단참고
      11. 사용자가 크기 조절 resize: none|both|horizontal|vertical|initial|inherit;
