  1. 브라-켓 표기법(Bra-ket notation; aka. 디랙 표기법)

    2. 켓-A : |A＞ = 열벡터 (A1, A2, ... An)

    브라-A : ＜A| = 행벡터 (A1*, A2*, ..., An*) // A*는 A의 켤레 복소수

  2. Qubit
    • |0＞, |1＞의 선형 결합 α|0＞+β|1＞
    • |0＞로 관측될 확률 |α|^2
    • |1＞로 관측될 확률 |β|^2
    • |α|^2 + |β|^2 = 1

    큐비트는 0인 동시에 1일 수 있으므로, 1큐비트는 1비트보다 2배 빠르게 연산이 가능하다

    동시 처리 가능한 비트 수의 증가는 선형적인 성능 향상을 가져오지만, 큐비트 수의 증가는 지수적인 성능 향상을 가져온다

  3. Superposition

    4. 입자는 동시에 여러 장소에 존재할 수 있다

  4. 양자 중첩 Quantum superposition

    5. 큐비트는 관측하기 전까지 어떤 상태인지 알 수 없다. 큐비트의 복사도 불가능하다

  5. 양자 얽힘 Quantum entanglement

    6. 얽혀있는 두 큐비트는 둘 사이의 거리에 무관하게 한쪽이 관측되면 다른 한쪽도 상태가 고정된다. 양자 얽힘에 의한 정보 전달은 빛보다 빠르지는 않다

    얽힌 두 큐비트를 얻는 방법 : 하나의 입자를 두 큐비트로 쪼개거나, 두 큐비트를 가까이 모아 얽힘 상태로 변경

  6. Quantum teleportation

    7. 양자 얽힘을 이용한 정보 전송 방법

    1. Alice와 Bob은 얽혀 있는 큐비트 한 쌍의 각각을 갖고 있다(Qa, Qb)
    2. Alice는 Bob에게 전달하고 싶은 큐비트(Qm)과 Qa를 동시와 관측한 뒤, 그 관측 결과를 Bob에게 기존 통신망으로 전달한다
    3. Bob은 전달받은 관측 결과와 Qb를 이용해 Qm을 알아낸다
