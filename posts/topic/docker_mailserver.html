설치 절차

  1. 설치 디렉터리에 다음 파일들 준비
    • docker-compose.yml
    • mailserver.env
    • setup.sh

      • 명령어 설명 :setup

  2. 루트 이메일 계정 추가

    3. ↓shell

    $ ./setup.sh email add name@example.com password
  3. DKIM 생성

    4. ↓shell

    $ ./setup.sh config dkim

    생성된 docker-data/dms/config/opendkim/keys/${domain}/mail.txt에 적힌 4개 줄

    ↓DKIM 예

    mail._domainkey IN TXT ( "v=DKIM1; h=sha256; k=rsa; " "p=MIICIjANBgkqhkiG9...bcg9S8pPBMP6UvHja" "CfmAAKOPddfdWGpzg5V...lua+sZuXwaAqBWjo" "b3DXduFMNIqaQ4Mgqn/...ZbKskd9ZUCAwEAAQ==" ) ; ----- DKIM key mail for 4joy.link
  4. DNS 설정
    5. 레코드 이름유형값
    4joy.linkMX1 mail.4joy.link
    mail._domainkey.4joy.linkTXTv=DKIM1; h=sha256; k=rsa; "
    "p=MIICIjANBgh...g9S8pPBMP6UvHja"
    "CfmAAKOPddf...sZuXwaAqBWjo"
    "b3DXduFMNI...9ZUCAwEAAQ=="')
    mail.4joy.linkAxxx.xxx.xxx.xxx
  5. 컨테이너 실행

    6. ↓shell

    $ docker-compose up -d --build
