FP 정의 from 『스칼라로 배우는 함수형 프로그래밍』

  1. Side effect 없는 순수 함수들로 앱을 만드는 것
  2. 순수 함수 : 반환값이 입력으로만 결정되는 함수. 입력이 불변하면 반환값도 불변
  3. f(x)가 참조에 투명한 모든 x에 대해 참조 투명하면 f는 순수 함수다
  4. 참조에 투명하다 : 어떤 표현식을 그 평가 결과로 바꾸어도 의미가 변하지 않는 경우

용어

  1. Currying; 커링

    2. 다인자 함수를 일인자 함수들의 체인으로 변경하는 방법; 다인자 함수를 적은 수의 파라미터로 호출하면 누락된 파라미터를 인자로 받는 함수를 반환

  2. Partial Application; 부분 적용, 부분 함수

    3. 다인자 함수의 인자들에 기본값을 주어 함수의 인자를 줄이는 방법

  3. HOF; higher-order function

    4. 고차 함수. 함수를 인자로 받는 함수

  4. Fold, Reduce, Accumulate, Aggregate, ...

    5. 재귀적 자료형에 대한 연산을 재귀적인 고차 함수로 정의하는 것; 또, 그렇게 정의한 함수
